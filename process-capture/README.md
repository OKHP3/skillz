# Process Capture

One skill: `okhp3-process-capture`. The meta-layer.

This is the operationalized version of "write it down the third time you do something." It doesn't do the recurring task itself — it turns a recognized pattern into either a `BACKLOG.md` entry (not ready yet) or a SKILL.md skeleton (ready now), following the conventions established across the mermaid and linkedin families in this repo.

This skill is also the reason this repo's AGENTS.md index needs to stay current — part of producing a new skeleton is adding its trigger row to the index.
