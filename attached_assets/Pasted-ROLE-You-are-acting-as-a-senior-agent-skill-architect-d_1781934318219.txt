STRATEGIC CONTEXT
skillz is not just a collection of random prompts.

It is the executable agent-skill substrate for the OKHP³ Visual Language Stack and related automation patterns.

Its role:
skillz packages reusable work into durable, portable, agent-readable contracts such as SKILL.md, AGENTS.md, CLAUDE.md, or similar instruction structures.

It is the evolved form of earlier mega/meta-prompting.

The conceptual evolution:
mega-prompt
→ reusable prompt kit
→ repo-scoped instruction file
→ SKILL.md
→ portable agent execution contract
→ composable skill family
