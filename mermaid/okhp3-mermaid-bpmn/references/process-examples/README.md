# Process Examples

This directory is reserved for synthetic, validated examples. A file may be added only after it records the intended audience, process boundary, Mermaid renderer/version, and Gate 1/2/3 result. Do not use real employer, customer, or private process data in examples.

## Suggested fixtures

- `approval-flow.mmd` - Analyst-tier XOR approval with a rejected exception path.
- `onboarding.mmd` - Executive-tier swim lanes with role handoffs and a terminal outcome.
- `procurement.mmd` - Technical-tier parallel review with an explicit AND join.

Until those fixtures exist, treat the patterns in the adjacent reference files as instructional guidance rather than known-good rendered artifacts.
