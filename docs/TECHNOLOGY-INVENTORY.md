# Technology Inventory

Audit date: 2026-07-13.

## Scope and method

This is a SKILL.md repository, not a deployed TypeScript, Vite, Tailwind, or React application. The inventory distinguishes technology required to execute repository scripts from technology shown only in examples, templates, or reference snippets. Version evidence comes from committed manifests, runtime configuration, source imports, and the latest official registry metadata.

## Maintenance plan

1. Dependabot opens weekly pull requests for the explicit npm, Python, and GitHub Action dependencies.
2. The `Refresh technology inventory` workflow runs every Monday and creates one pull request when Node LTS, Python, or any observed latest release changes.
3. Review generated-project guidance separately before changing Vite, React, or Tailwind. Those are examples, not live dependencies, and Tailwind 4 requires template changes.
4. Merge only after normal repository validation passes. The automation has no auto-merge permission.

For scheduled pull requests to work, the repository's Actions settings must allow `GITHUB_TOKEN` to create pull requests.
